# brainsaver

BrainVault — semplice prototipo client-side per catturare e organizzare note, link, file e registrazioni vocali. I dati sono salvati nel `localStorage` del browser.

Quick start:

1. Apri [index.html](index.html) in un browser moderno (o servi la cartella con un semplice server statico).
2. Crea cartelle, aggiungi note/link/file e prova le funzioni di upload e riconoscimento vocale (se supportato dal browser).

Suggerimenti per sviluppo:

- Separare ulteriormente componenti, aggiungere build step (Vite/Parcel) per produzione.
- Aggiungere backend per persistenza e autenticazione.
- Introdurre linting: `eslint` + `stylelint`.
